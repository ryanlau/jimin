starter file

Begin by copying and pasting the following code into an HTML file.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Bootstrap demo</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-rbsA2VBKQhggwzxH7pPCaAqO46MgnOM80zW1RWuH61DGLwZJEdK2Kadq2F9CUG65" crossorigin="anonymous">
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Include Bootstrap's js features by adding the following line to the bottom of the HTML body.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-kenU1KFdBIe4zVF0s0G1M5b4hcpxyD9F7jL+jjXkk+Q2h455rYXK/7HAuoJl+0I4" crossorigin="anonymous"></script>

grid

Children of a element with the container class become grid elements. Bootstrap's grid system allows us to align and place elements. A grid is comprised of elements with the row class. Within these elements, there are elements with the col class that make up each item of the grid.

The grid above is the result of the following code:

<div class="container">
  <div class="row">
    <div class="col bg-danger">a</div>
    <div class="col bg-success">b</div>
    <div class="col bg-primary">c</div>
  </div>
</div>

By default, all columns of a row have the same width. To have elements of different widths, the col-{1-12} class can be used in place of col. col-5 means that element will be 5/12 the width of the container element.

spacing

You can easily add padding or margin to an element with the provided padding and margin classes.

To add padding to an element, apply class p{t|b|l|r|x|y}-{0-5} to the element. To add margin to an element, apply class m{t|b|l|r|x|y}-{0-5} to the element.

options

  • t: apply spacing only on top
  • b: apply spacing only on bottom
  • l: apply spacing only on the left
  • r: apply spacing only on the right
  • x: apply spacing only on the x-axis (left and right)
  • y: apply spacing only on the y-axis (top and bottom)
  • 0: no spacing
  • 5: most amount of spacing

As an example, mb-0 removes the margin from the bottom of the element.
